# DATA STRUCTURES Documentation

## BASIC Variables

### Integer
- **Description:** Represents whole numbers.
- **Example:** `Dim age As Integer = 25`

### Float
- **Description:** Represents numbers with decimals.
- **Example:** `Dim temperature As Float = 36.6`

### String
- **Description:** Represents sequences of characters.
- **Example:** `Dim name As String = "John Doe"`

### Boolean
- **Description:** Represents logical true or false values.
- **Example:** `Dim isActive As Boolean = True`

## Record Structures

### Student Record
- **Description:** Represents a student's information.
- **Fields:**
  - **Name:** String
  - **Age:** Integer
  - **Grade:** String

### Employee Record
- **Description:** Represents an employee's information.
- **Fields:**
  - **EmployeeID:** Integer
  - **Name:** String
  - **Position:** String
  - **Salary:** Float

### Product Record
- **Description:** Represents a product's details.
- **Fields:**
  - **ProductID:** Integer
  - **ProductName:** String
  - **Price:** Float
  - **InStock:** Boolean

## Conclusion
This documentation provides an overview of basic variable types and record structures necessary for effective programming.
