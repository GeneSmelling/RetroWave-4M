# Development Setup and Installation Instructions

## Prerequisites
- Ensure you have [Node.js](https://nodejs.org/) installed.
- Ensure you have [npm](https://www.npmjs.com/) installed.

## Installation Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/GeneSmelling/RetroWave-4M.git
   cd RetroWave-4M
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Running the Project
- To start the project, run:
   ```bash
   npm start
   ```

## Running Tests
- To run the tests, execute:
   ```bash
   npm test
   ```

## Additional Info
- For further configuration options, check the documentation in the `docs/` folder.
